Pantalla OLED

José Sebastián De los Santos Hernández
Ing. Sistemas computacionales
Fecha: Noviembre 16, 2023
Arduino Electrónica Microcontroladores

- Resumen -

Resumiendo la práctica, esta consiste en conectar una pantalla OLED, la cual sigue el protocolo i2c, el cual es de comunicación y nos permite conectar mas de un dispositivo o elemento a los mismos puertos, dado esto necesitabamos imprimir en la pantalla ON u OFF según la señal recibida por un boton.

Esto con ayuda de una protoboard, cables y algunos otros materiales que serán mencionados mas adelante.

- Introducción -

Introducciendo esta práctica, se exploró la interacción entre una pantalla OLED, un botón, y el microcontrolador ATtiny85, utilizando la placa de desarrollo Arduino UNO como plataforma central. El objetivo principal consistió en imprimir en la pantalla OLED el estado de la señal generada por un botón. Para lograr esto, se implementó una conexión eficiente entre los elementos mediante el microcontrolador ATtiny85, aprovechando la versatilidad y capacidad de procesamiento de la placa Arduino UNO como base para el desarrollo del proyecto. Este enfoque permitió crear un sistema integrado que respondía de manera visual a las acciones del usuario a través del botón.

Como conceptos básicos, tenemos el protocolo i2c, este como tal, no se implementó en la práctica, pero es importante en caso de querer hacer un circuito mas grande con el mismo micro. Pues con ayuda de este protocolo se puede generar una comunicación entre dos microcontroladores y asi poder hacer un circuito mas preparado y grande.

- Materiales -

Se utilizaron distintos materiales siendo en su mayoria Outputs.

  • Protoboards
  • Resistencias de 10kΩ
  • Botones de 2 pines
  • Jumpers Macho-Macho
  • Placa Arduino UNO
  • Microcontrolador Attiny85
  • Pantalla OLED

- Desarrollo -

El proceso no fue muy tardado ni tan dificil, pues realmente bastaba con conectar la pantalla los pines SDA Y SCL a los pines con los mismos datos en nuestro microcontrolador, y un boton para el funcionamiento completo de nuestro sistema al pin que queramos.

Simulación

En este caso, la simulación será especificamente de la pantalla funcionando con un sensor de temperatura, pero para nuestro circuito seguimos la misma lógica de armado y de programación.

Diseño

A continuación, se mostrará el diseño del código, en lenguaje C.

- Resultados -

Se logró un resultado optimo, pues se puede observar que la pantalla muestra algo segun la señal que proporciona el boton.

- Conclusiones -

En conclusión, la práctica destacó la integración exitosa entre una pantalla OLED, un botón y el microcontrolador ATtiny85, utilizando la plataforma Arduino UNO como base. Esto permitió imprimir de manera efectiva en la pantalla el estado de la señal generada por el botón. El ejercicio resaltó la versatilidad del entorno de desarrollo Arduino para la creación de sistemas interactivos y visuales. Además, la utilización del ATtiny85 evidenció cómo microcontroladores de menor escala pueden desempeñar un papel crucial en proyectos que requieren eficiencia y economía de recursos. En resumen, la práctica proporcionó una valiosa experiencia práctica en la integración de componentes electrónicos para lograr un sistema funcional y visualmente informativo.
